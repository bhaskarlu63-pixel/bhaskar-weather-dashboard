# 🌦 Bhaskar Weather Dashboard

A modern, responsive weather web application that provides real-time weather updates, location-based detection, and detailed forecasts with a clean glassmorphism UI.

---

🚀 Live Demo
🔗 https://bhaskar-weather-dashboard.netlify.app/

---

📌 Project Overview
This project is a fully functional weather dashboard built using pure frontend technologies. It integrates with the OpenWeather API to fetch real-time weather data and display it in a visually appealing and user-friendly interface.

The app supports automatic location detection as well as manual city search, making it versatile and easy to use.

---

✨ Features

🌍 Core Features
- 📍 Auto-detect user location (Geolocation API)
- 🔍 Search weather by city name
- 🌡 Real-time temperature display
- 🌥 Weather condition description
- 💧 Humidity information
- 🌬 Wind speed details
- ⚡ Atmospheric pressure

⏱ Forecast Features
- ⏰ Hourly weather forecast
- 📅 Tomorrow weather prediction
- 🌅 Sunrise & Sunset timing

🎨 UI/UX Features
- 🧊 Glassmorphism design
- 🎨 Dynamic background based on weather
- 📱 Fully responsive (mobile-friendly)
- ✨ Smooth animations
- 🌈 Clean and modern layout

🧠 How It Works

1. The app uses the **Geolocation API** to detect user location.
2. It sends requests to the **OpenWeather API**.
3. Receives JSON weather data.
4. Updates UI dynamically using JavaScript.
5. Displays current, hourly, and forecast data.
